# Hastebin shell function
Forked from https://github.com/diethnis/standalones
## How to use
### with file
```
$ cat file.log | haste
# or
$ haste file.log
```
### with direct input
```
$ haste
blahblahblah
(Ctrl+D)
```

## Install
I added this "hastebin.sh" to my `/etc/profile.d/`. Must restart shell to see the changes.
